Quick object dump with the impure JsonPipe.
Impure: Changes to the object will automatically update, not just reference changes. {{ {number: 7} | json }}
{{ 'LOWERCASE' | lowercase }} {{ 'uppercase' | uppercase }} {{ 'tHIs is tiTLE CaSe' | titlecase }}

Works on a Date, a number (ms since UTC epoch), or an ISO string

Examples

Default: mediumdate format, local system timezone and undefined locale.
Only en-US locale comes shipped with Angular.
{{ Date.parse('2019-05-01') | date }} = 1 mai 2019
{{ Date.parse('2019-05-01T23:55:00') | date:'yyyy-MM-dd HH:mm:ss' }} = 2019-05-01 23:55:00

More options

  • Week of year: w=7, ww=07
  • Week day: E=Tue, EEEE=Tuesday, EEEEE=T, EEEEEE=Tu
  • Month standalone: LLLL=September
  • Period: a=am/pm
  • Hour 1-12: h=1, hh=01
  • Zone: z=GMT-8, Z=-0800
    • Or named: short, medium, long, full; shortDate, mediumTime, ...

    Timezone and locale

    {{ Date.parse('2019-05-01T23:55:00') | date:'yyyy-MM-dd HH:mm:ss':'-0055':'fr' }} = 2019-05-01 21:00:00

    CurrencyPipe

  • currencyCode: USD, EUR, GBP, ...
  • display: code=USD, symbol=$, a string
